# Week12 실습

## 오늘 실행해본 코드 가장 흥미로웠던 것
bsp_dungeon을 실행해봤을 때 던전의 복잡하고 랜덤한 구조가 스페이스를 누르면 프로시저널을 활용해 만들어지는 과정을 본 것이 신기해서 가장 흥미로웠다.

## 파라미터를 바꿨을 때 생기는 변화
bsp_dungeon에서 파라미터를 바꾸면 bsp 던전의 구조, 방 크기, 통로 형태, 생성 속도, 화면 배치 등이 크게 달라졌다.

## 기말 프로젝트에 프로시저럴 생성을 활용할 수 있는가?
프로시저널을 잘 활용할 수 있는 게임을 기말 프로젝트로 만든다고 생각하면 충분히 좋게 활용할 수 있을 거 같다.
